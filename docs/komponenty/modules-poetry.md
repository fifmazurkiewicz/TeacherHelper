# modules/poetry — generator wierszy

## Odpowiedzialność

Wiersze do recytacji: temat, forma, długość, nastrój, podział na głosy.

## Formaty

PDF (układ ozdobny), DOCX, TXT.

## Adapter

**OpenRouter** (tekst); formatowanie i szablony mogą być w warstwie export. Uruchamiany przez tool call `generate_poetry`.
